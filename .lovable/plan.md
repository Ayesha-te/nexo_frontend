

# Nexocart - Binary Platform

A green & gold themed platform with user dashboard and admin panel, using mock/static data.

## Pages & Features

### 1. Login Page
- Email + password login form
- Nexocart branding with green & gold theme
- Mock login (any credentials work)

### 2. User Dashboard
- **Stats Cards**: Current Income, Reward Income, Total Withdraw, Available Pins, Left Team count, Right Team count
- Green & gold color scheme with modern card layouts

### 3. Sidebar Navigation (User)
- Dashboard, Pin Code Request, My Pins, Add New User, My Tree, Withdraw History, Change Password, Profile Setting, Logout

### 4. Pin Code Request Page
- Display admin payment info (EasyPaisa: 03448252109, Sardar Laeiq Ahmed)
- Form: Account Number, TRX ID, Amount
- Urdu description about pin token purchase process
- Request submission with pending status

### 5. My Pins Page
- List of purchased pin tokens with status (used/available)
- Copy pin token functionality

### 6. Add New User (Account Activation)
- Form: Pin Token (paste), First Name, Last Name, Email, Account Number, Referral Email, Position (Left/Right), Payment Method (EasyPaisa/JazzCash)
- Auto-generated password = phone number
- Validation for all fields

### 7. My Tree (Binary Tree View)
- Visual binary tree showing left & right team members
- Show user names and levels

### 8. Withdraw History
- Table of daily auto-withdrawals
- Show 5% normal tax, 10% on cap limit & rewards
- Cap limit: 4,000 PKR

### 9. Income System (displayed in dashboard)
- Referral income: First pair = 400, remaining pairs = 200
- Earnings stop at 50k pairs (1 lakh members) → fixed salary 50k/month
- **Rewards table** with 10 levels (25/25 → Certificate up to 50k/50k → 1,500,000)

### 10. Profile Settings
- Editable: Profile picture, Name
- View only: Email, Phone (changeable by admin only)

### 11. Change Password Page

### 12. Admin Panel
- **Admin Dashboard**: Overview stats, pending pin requests, total users
- **Manage Pin Requests**: Approve/reject pin token requests with verification
- **Manage Users**: View all users, edit email/phone, activate/deactivate
- **Withdrawal Management**: View daily auto-withdrawals, process payments
- **Admin login** separate from user login
